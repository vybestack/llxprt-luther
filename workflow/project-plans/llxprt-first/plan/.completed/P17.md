## Step Inventory

1. `select_issue` - shell: Query GitHub for open issue, assign it
2. `setup_workspace` - shell: Clone/fetch repo, create branch, create .luther/
3. `fetch_issue` - shell: Fetch issue body/comments, write to .luther/ files
4. `create_plan` - shell: Invoke llxprt with planning profile
5. `evaluate_plan` - shell: Invoke llxprt to review plan
6. `implement` - shell: Invoke llxprt to write code
7. `evaluate_impl` - shell: Invoke llxprt to review implementation
8. `run_tests` - verify: Run lint, typecheck, test, format, build
9. `remediate` - shell: Invoke llxprt to fix test failures
10. `push_changes` - shell: git add, commit, push
11. `generate_pr_description` - shell: Invoke llxprt to write PR description
12. `create_pr` - shell: gh pr create with --body-file
13. `abandon_and_log` - shell: Comment on issue, remove label, unassign
14. `log_completion` - shell: Record success outcome
