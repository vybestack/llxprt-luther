When parsing fails, all parsers wrap raw output in structured ErrorRecord:
```rust
ErrorRecord {
    file: None,
    line: None,
    column: None,
    message: raw_output.trim().to_string(),
    severity: Some("error".to_string()),
    test_name: None,
    assertion_kind: None,
    expected: None,
    actual: None,
}
```

CheckResult always contains `check_type`, `exit_code`, `raw_stdout`, `raw_stderr`.
