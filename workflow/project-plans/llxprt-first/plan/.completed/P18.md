## Design Features

### ConfigurableExecutor
- `SharedMockExecutor` stores `HashMap<String, Vec<StepOutcome>>`
- Registered for both "shell" and "verify" step types
- Tracks call counts to verify loop iterations
- Returns Success for steps not in map

### Live Test Design
- All repo-specific values loaded from TOML config fixture
- NO hardcoded repo names, org names, or profile names
- Uses `resolve_workflow_config()` to read `target_repo`, `base_branch`, `assignee`
- Uses `tempfile::TempDir` for workspace tests
- Structural assertions only (no specific content assertions)
